## 1 元数据分类

| 类别 | 内容 | 存储位置 | 管理工具 |
|------|------|---------|---------|
| **技术元数据** | 库/表/字段/分区/存储格式 | MySQL `meta_table_def`, `meta_column_dict` | 元数据管理系统 |
| **业务元数据** | 字段中文名/业务含义/码值映射/数据字典 | MySQL `meta_column_dict` | 数据字典 |
| **操作元数据** | ETL作业/调度依赖/运行日志/执行时长 | MySQL `meta_job_log` | DolphinScheduler |
| **映射元数据** | 源表→目标表映射/字段血缘 | MySQL `meta_lineage` | 血缘分析 |

## 2 元数据表设计

### 2.1 meta_table_def — 表定义

```sql
CREATE TABLE meta_table_def (
    table_id        INT PRIMARY KEY AUTO_INCREMENT,
    table_schema    VARCHAR(100) NOT NULL,   -- dwd/dws/ads/dim
    table_name      VARCHAR(200) NOT NULL,   -- dwd_hub_customer
    table_cn_name   VARCHAR(200),            -- 客户中心表
    table_type      VARCHAR(50),             -- HUB/LINK/SAT/FACT/DIM
    topic_domain    VARCHAR(50),             -- 客户域/存款域
    storage_format  VARCHAR(20) DEFAULT 'ORC',
    partition_key   VARCHAR(100),            -- dt
    bucket_key      VARCHAR(100),
    bucket_num      INT,
    retention_days  INT,
    owner           VARCHAR(100),
    created_date    DATE,
    updated_date    DATE,
    description     TEXT
);
```

### 2.2 meta_column_dict — 字段字典

```sql
CREATE TABLE meta_column_dict (
    column_id       INT PRIMARY KEY AUTO_INCREMENT,
    table_id        INT REFERENCES meta_table_def(table_id),
    column_name     VARCHAR(200) NOT NULL,
    column_type     VARCHAR(50),
    column_cn_name  VARCHAR(200),           -- 客户姓名
    is_pk           BOOLEAN DEFAULT FALSE,
    is_fk           BOOLEAN DEFAULT FALSE,
    ref_table       VARCHAR(200),
    code_dict       TEXT,                   -- JSON: {"01":"活期","02":"定期"}
    nullable        BOOLEAN DEFAULT TRUE,
    sample_value    VARCHAR(500),
    description     TEXT
);
```

### 2.3 meta_job_log — ETL作业日志

```sql
CREATE TABLE meta_job_log (
    job_instance_id VARCHAR(100) PRIMARY KEY,
    job_name        VARCHAR(200) NOT NULL,
    job_type        VARCHAR(50),            -- DATAX/SPARK/HIVE/FLINK
    schedule_date   DATE,
    start_time      TIMESTAMP,
    end_time        TIMESTAMP,
    duration_sec    INT,
    status          VARCHAR(20),            -- RUNNING/SUCCESS/FAILED/TIMEOUT
    rows_processed  BIGINT,
    error_msg       TEXT,
    retry_count     INT DEFAULT 0
);
```

## 3 元数据应用

| 应用场景 | 使用方式 |
|---------|---------|
| **影响分析** | 修改某表→查询meta_lineage→得到所有下游表 |
| **数据字典查询** | 业务人员查meta_column_dict了解字段含义 |
| **资产盘点** | 统计meta_table_def中的表数量、存储量、负责人 |
