```
源系统表 → STG → ODS → DWD(Hub/Link/Sat) → DWS(维度/事实) → ADS → 应用
```

## 3 血缘表设计

```sql
CREATE TABLE meta_lineage (
    lineage_id      INT PRIMARY KEY AUTO_INCREMENT,
    source_system   VARCHAR(50),         -- 源系统
    source_table    VARCHAR(200),        -- 源表
    source_column   VARCHAR(200),        -- 源字段
    target_schema   VARCHAR(100),        -- 目标层: dwd/dws/ads
    target_table    VARCHAR(200),        -- 目标表
    target_column   VARCHAR(200),        -- 目标字段
    transform_rule  TEXT,                -- 转换规则(SQL片段)
    lineage_type    VARCHAR(50),         -- COPY/TRANSFORM/AGGREGATE/JOIN
    etl_job_name    VARCHAR(200),        -- 产生血缘的ETL作业
    etl_time        TIMESTAMP
);
```

---

## 4 血缘应用

- **影响分析**：修改某源表 → 递归查询下游所有依赖
- **问题溯源**：报表数据异常 → 反向追踪到源表
- **合规审计**：证明监管报送数据来源可追溯
