## 1 数据分级

| 级别 | 标识 | 内容 | 示例 |
|------|------|------|------|
| **L0 公开** | PUBLIC | 可对外公开的数据 | 产品利率、营业网点地址 |
| **L1 内部** | INTERNAL | 内部使用，不外传 | 汇总统计数据、经营分析报告 |
| **L2 敏感** | SENSITIVE | 客户个人敏感信息 | 姓名、手机号、地址、身份证号 |
| **L3 绝密** | CONFIDENTIAL | 核心机密数据 | 客户完整账号、交易密码、卡CVV |

## 2 脱敏规则

| 字段 | 脱敏方式 | 原始值 | 脱敏后 |
|------|---------|--------|--------|
| 客户姓名 | 保留姓，名用* | 李明华 | 李\*\* |
| 身份证号 | 保留前3后4 | 310123199001011234 | 310\*\*\*\*\*\*\*\*\*\*\*\*1234 |
| 手机号 | 保留前3后4 | 13812345678 | 138\*\*\*\*5678 |
| 账号 | 保留前4后4 | 6222021234567890 | 6222\*\*\*\*\*\*7890 |
| 卡号 | 保留前6后4 | 6228481234567890 | 622848\*\*\*\*\*\*7890 |
| 地址 | 只保留到区/县 | 北京市朝阳区望京街道xxx | 北京市朝阳区\*\*\*\* |
| 余额 | 区间化 | 123,456.78 | 10万-50万 |

## 3 脱敏实现

```sql
-- Hive UDF: mask_name
CREATE FUNCTION mask_name AS 'com.bank.udf.MaskNameUDF';
-- UDF: mask_id_no
CREATE FUNCTION mask_id_no AS 'com.bank.udf.MaskIdNoUDF';
-- UDF: mask_mobile
CREATE FUNCTION mask_mobile AS 'com.bank.udf.MaskMobileUDF';

-- 创建脱敏视图
CREATE VIEW dwd_v_customer_masked AS
SELECT
    customer_hash_key,
    mask_name(cust_name) AS cust_name,
    mask_id_no(id_no) AS id_no,
    mask_mobile(mobile) AS mobile,
    cust_type, cust_level, gender, province, city  -- 非敏感字段保留
FROM dwd_sat_customer_info
WHERE is_current = TRUE;
```

## 4 访问控制

| 角色 | 权限 | 说明 |
|------|------|------|
| `分析师` | 脱敏视图查询 | 只能看 L0-L1 + L2脱敏数据 |
| `开发者` | 全量表+脱敏视图 | 开发测试用，可看全量 |
| `管理员` | 全部数据 | 运维管理，可看L3 |
| `审计员` | 元数据+审计日志 | 只能看操作记录 |

## 5 审计日志

```sql
-- HiveServer2审计日志 (hive.server2.audit.enabled=true)
-- 记录: 谁(user)在何时(timestamp)从哪个IP查询了哪些表

CREATE TABLE audit_query_log (
    log_id          BIGINT,
    query_user      STRING,
    query_time      TIMESTAMP,
    client_ip       STRING,
    operation       STRING,        -- QUERY/INSERT/CREATE/DROP
    database_name   STRING,
    table_name      STRING,
    query_sql       STRING,
    rows_returned   BIGINT,
    duration_ms     BIGINT
)
PARTITIONED BY (dt STRING)
STORED AS ORC;
```
