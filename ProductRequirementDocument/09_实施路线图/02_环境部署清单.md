## 2 待部署组件（按优先级）

### 高优先级（一期必备）

#### Apache Kafka
```bash
# 下载与安装
wget https://archive.apache.org/dist/kafka/3.9.0/kafka_2.13-3.9.0.tgz
tar -xzf kafka_2.13-3.9.0.tgz -C /mnt/d/bigdata-lab/software/
export KAFKA_HOME=/mnt/d/bigdata-lab/software/kafka_2.13-3.9.0

# 启动 (单节点，用于开发学习)
$KAFKA_HOME/bin/zookeeper-server-start.sh $KAFKA_HOME/config/zookeeper.properties &
$KAFKA_HOME/bin/kafka-server-start.sh $KAFKA_HOME/config/server.properties &
```

#### Canal
```bash
wget https://github.com/alibaba/canal/releases/download/canal-1.1.7/canal.deployer-1.1.7.tar.gz
tar -xzf canal.deployer-1.1.7.tar.gz -C /mnt/d/bigdata-lab/software/canal
# 配置后启动
/mnt/d/bigdata-lab/software/canal/bin/startup.sh
```

#### Redis
```bash
docker run -d --name redis -p 6379:6379 redis:7-alpine
```

### 中优先级（二期）

#### DolphinScheduler
```bash
wget https://archive.apache.org/dist/dolphinscheduler/3.2.1/apache-dolphinscheduler-3.2.1-bin.tar.gz
tar -xzf apache-dolphinscheduler-3.2.1-bin.tar.gz -C /mnt/d/bigdata-lab/software/
```

### 低优先级（三期/可选）

- Flume：文件采集（ATM/POS模拟）
- HBase：实时明细存储
