## 1 日批时间线 (T+1)

```
时间        阶段              作业内容                       并行度
────────────────────────────────────────────────────────────
00:00     STG采集        DataX全量+增量抽取 11个源系统       全部并行
          预计耗时: 30min
          
00:30     STG清理        删除7天前分区                      一
          
01:00     ODS贴源        STG→ODS, 类型转换+元数据添加       按源系统并行
          预计耗时: 30min
          
02:00     DWD Hub        生成HASH_KEY, 识别新实体           所有Hub并行
          预计耗时: 30min    (客户/账户/合同/产品/机构/交易...)

03:00     DWD Satellite  属性拉链更新 + 事实追加            所有Sat并行
          预计耗时: 30min    (客户信息/账户余额/合同信息...)

03:30     DWD Link       关系提取                           所有Link并行
          预计耗时: 30min    (客户-账户/贷款-放款/交易参与方)

04:00     DWD Fact       事务事实表                         按域并行
          预计耗时: 60min    (交易统一事实/还款明细/理财交易)

05:00     DWS汇总        维度表 + 汇总事实表                 按域并行(8域)
          预计耗时: 90min    (客户/存款/贷款/支付/信用卡/理财/风控/财务)

06:30     DQ质量检查     完整性/唯一性/一致性/参照完整性      全部并行
          预计耗时: 30min

07:00     ADS应用层      管理驾驶舱/监管报送/客户画像/BI      按应用并行(5应用)
          预计耗时: 30min

07:30     统计信息更新    ANALYZE TABLE统计信息              按表顺序
          预计耗时: 15min

07:45     通知与归档      钉钉/邮件通知 + 作业日志归档         一
          预计耗时: 5min

08:00     日批完成       总耗时: 约8小时
```

## 2 非日批作业

### 2.1 小时批 (每小时整点)

```sql
-- 风控指标准实时更新 (某些DWS表每小时刷新)
INSERT OVERWRITE TABLE dws_risk_daily_monitor PARTITION (dt='${today}')
SELECT ... WHERE dt = '${today}';
```

### 2.2 周批 (每周一)

```sql
-- 周度汇总表
INSERT OVERWRITE TABLE dws_weekly_summary PARTITION (week='${last_week}')
SELECT ...;
```

### 2.3 月批 (每月1日)

```sql
-- 月度汇总 + 结息 + 月报
INSERT OVERWRITE TABLE ads_reg_1104_g01 PARTITION (report_month='${last_month}')
SELECT ...;
```
