## 1 BI分析主题

| 分析主题 | 预置分析场景 | 消费表 |
|---------|------------|--------|
| **存款分析** | 结构分析、流失预警、利率敏感性、到期预测 | `ads_bi_deposit_wide` |
| **贷款分析** | 投放趋势、还款预测、利率分析、逾期监控 | `ads_bi_loan_wide` |
| **客户分析** | 分层透视、活跃度、交叉销售、生命周期 | `ads_cust_360_view`, `ads_cust_segment` |
| **渠道分析** | 渠道对比、替代率、客户偏好、时效 | `ads_bi_channel_wide` |
| **机构分析** | 网点对标、绩效排名、产能评估 | `ads_mgmt_branch_ranking` |

## 3 典型分析SQL

### 3.1 存款流失预警

```sql
SELECT 
    cust_segment,
    COUNT(CASE WHEN balance_change_30d < -0.3 THEN 1 END) AS churn_risk_cust,
    SUM(CASE WHEN balance_change_30d < -0.3 THEN balance ELSE 0 END) AS at_risk_amount
FROM ads_bi_deposit_wide
WHERE dt = '${yesterday}'
GROUP BY cust_segment;
```

### 3.2 渠道替代率分析

```sql
SELECT 
    dt,
    channel_name,
    txn_cnt,
    ROUND(txn_cnt * 100.0 / SUM(txn_cnt) OVER(PARTITION BY dt), 2) AS share_pct,
    LAG(txn_cnt, 30) OVER(PARTITION BY channel_name ORDER BY dt) AS txn_30d_ago,
    ROUND((txn_cnt - LAG(txn_cnt, 30) OVER(...)) * 100.0 / LAG(txn_cnt, 30) OVER(...), 2) AS growth_30d_pct
FROM ads_bi_channel_wide
WHERE dt >= DATE_SUB('${yesterday}', 30);
```

### 3.3 客户交叉销售分析

```sql
SELECT 
    product_holding_cnt,
    COUNT(1) AS cust_cnt,
    ROUND(AVG(total_aum), 2) AS avg_aum,
    ROUND(AVG(value_score), 1) AS avg_value_score
FROM ads_cust_360_view
WHERE dt = '${yesterday}'
GROUP BY product_holding_cnt
ORDER BY product_holding_cnt;
```
