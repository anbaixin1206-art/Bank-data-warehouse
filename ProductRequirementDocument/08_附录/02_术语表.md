## 1 数仓术语

| 术语 | 英文 | 说明 |
|------|------|------|
| STG | Staging | 临时缓冲层，原始数据落地 |
| ODS | Operational Data Store | 操作数据层，贴源存储 |
| DWD | Data Warehouse Detail | 明细数据层，清洗整合后的明细 |
| DWS | Data Warehouse Summary | 汇总数据层，轻度汇总 |
| ADS | Application Data Store | 应用数据层，面向应用的大宽表 |
| Hub | Hub Table | Data Vault中心表，存储业务实体标识 |
| Link | Link Table | Data Vault链接表，存储实体间关系 |
| Satellite | Satellite Table | Data Vault卫星表，存储属性/事实 |
| SCD | Slowly Changing Dimension | 缓慢变化维度 |
| ETL | Extract-Transform-Load | 数据抽取-转换-加载 |
| CDC | Change Data Capture | 变更数据捕获 |
| CEP | Complex Event Processing | 复杂事件处理 |
| DQ | Data Quality | 数据质量 |
| HASH_KEY | Hash Key | Data Vault的MD5哈希主键 |
| HASH_DIFF | Hash Difference | 属性变更检测MD5值 |
